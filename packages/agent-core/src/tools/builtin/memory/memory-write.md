# MemoryWrite

Persist a fact, insight, decision, preference, or snippet to cross-session memory.

Use this when the user shares a durable piece of information that should be remembered across sessions — preferences, recurring workflows, project conventions, decisions, or reusable snippets. The memory is stored in `.kimi-code/memory/entries.json` and can be retrieved later with `MemoryRead`, `MemorySearch`, or auto-recalled via the Orchestrator.
